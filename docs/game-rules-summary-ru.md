# Letter: No Mercy — Обновлённая сводка правил для реализации

Документ объединяет:

- обновлённые правила из PDF `Letter_ No Mercy - Dark Medieval Stories.pdf`;
- ранее согласованные решения из `docs/game-rules-questions-ru.md` и `docs/game-rules-questions-round-2-ru.md`.

Если правило из PDF конфликтует с ранее согласованным, в этом документе сохраняется ранее согласованное правило, а конфликт выносится в блок “Нужны уточнения”.

## 1) Матч, раунды, победа

- Игра на 2-6 игроков.
- Матч состоит из последовательных раундов.
- Цель по ПО:
- 2 игрока: 7 ПО
- 3 игрока: 6 ПО
- 4 игрока: 5 ПО
- 5-6 игроков: 4 ПО

## 2) Подготовка

1. Убрать карту `Тень (10)` из колоды персонажей.
2. Перетасовать колоды персонажей, указов и событий.
3. Убрать одну случайную карту из колоды персонажей в отложенную зону раунда.
4. Раздать каждому игроку по 1 карте.
5. Если на старте у игрока `Знамение`, заменить его.
6. Определить первого игрока.
7. Первый игрок смотрит 2 верхние карты указов, выкладывает 1, вторую кладёт под низ колоды.
8. Подготовить жетоны: `Милость королевы`, `Знак феодала`, `Чёрная роза`.

## 3) Режим на двоих

- Каждый раунд раскрываются 3 верхние карты колоды персонажей.
- Эти карты только информационные:
- не участвуют в эффектах;
- их нельзя перевернуть, заменить, вернуть, удалить.

## 4) Термины

- Разыграть карту: сбросить карту и применить её эффект (или эффект указа, если указ заменяет эффект персонажа).
- Сбросить карту: положить карту в сброс игрока.
- Убрать карту/жетон: не класть в сброс, а отложить в сторону до следующего раунда.
- Значение карты: число в левом верхнем углу, может меняться эффектами.

## 5) Указы и ротация

- Указы действуют на матч.
- Для каждого персонажа одновременно действует только 1 указ.
- Общие указы можно иметь несколько (в пределах общего лимита карт указов на столе).

Ротация:

- Лидер раунда: игрок с наибольшим ПО; при равенстве — первый игрок или победитель прошлого раунда.
- Фаза добора: до 3 активных указов на столе.
- Фаза сопровождения: после достижения 3 указов лидер заменяет один из активных.
- При выборе указа: берутся 2 верхние, 1 выбирается, 1 уходит под низ.
- Дубликат указа того же персонажа заменяется (перетяжка).

## 6) Базовая модель хода (согласованная)

1. Игрок выбирает и разыгрывает карту.
2. Движок определяет итоговый эффект:
- учитывает активное событие;
- если есть указ персонажа, эффект указа заменяет базовый эффект карты.
3. Эффект применяется атомарно одним блоком (все сбросы/выбывания/изменения внутри этого блока).
4. После полного завершения эффекта — проверка конца раунда.
5. Если раунд не завершён — ход передаётся следующему возможному игроку.

## 7) События: приоритет и базовые правила

- Эффекты событий имеют наивысший приоритет.
- В раунде обычно 1 активное событие, но допускаются исключения (например, через `Братоубийство`).
- Ранее согласовано: технический лимит активных событий в раунде может быть 2; при срабатывании второго события его эффект также действует в этом раунде.

## 8) Знамение

- При вытягивании `Знамения` событие разыгрывается немедленно.
- `Знамение` не считается обязательным розыгрышем карты за ход.
- `Знамение` откладывается отдельно (не в сброс).
- После события игрок добирает карту и продолжает ход по обычному правилу.

## 9) Дворянка (защита)

Согласованная модель:

- защищает от эффектов карт других игроков, включая массовые эффекты “Все игроки” на картах игроков;
- не защищает от событий;
- не защищает от указов;
- forced discard от событий защитой не блокируется.

## 10) Ополчение (Крестьянин как Стражник)

- При `Ополчении` Крестьянин разыгрывается как Стражник.
- Если есть указ на Стражника, применяется указ Стражника.
- Для идентичности карты и угадывания карта остаётся Крестьянином.

## 11) Значения карт

- Модификаторы значений суммируются.
- Итоговое значение может быть ниже 0.
- Нижнего порога нет.
- Для сравнений используется фактическое итоговое значение.

## 12) Перевёрнутый сброс

- При `Пелене` / `Глухой тьме` чужой сброс скрывается визуально.
- Серверно карты остаются обычными картами сброса, просто скрыты для чужого UI.
- Свой сброс игрок видит.

## 13) Удаление карт из игры

- Удалённые карты исключаются из последующих раундов (не замешиваются обратно).
- Хранятся в открытой отдельной зоне.

## 14) Жетоны

### Знак феодала
- Действует до конца раунда.
- Учитывается при начислении ПО даже если носитель выбыл.
- Может быть более одного у игрока (практически до 2).

### Чёрная роза
- Ставится эффектом Епископа.
- Защищает от обязательного сброса карты со значением 7+ (включая эффекты карт/указов/событий).
- Если из-за модификаторов значение карты стало < 7, защита не срабатывает.
- При срабатывании жетон тратится.

### Милость королевы
- В игре одна.
- Не переходит другому игроку в том же раунде.
- По согласованному правилу: владелец Королевы получает 1 ПО, если цель жетона дожила до конца раунда.

## 15) События, влияющие на конец раунда

- `Эхо прошлого`: раунд завершается немедленно; участвуют все игроки (включая выбывших), сравнение по старшей карте сброса.
- Tie-break по согласованному правилу:
- равная старшая карта -> сравнение суммы сброса;
- если снова равенство -> все игроки получают по 1 ПО.
- `Равенство`: раунд немедленно завершается, оставшиеся в игре получают ПО по тексту события.
- `Перерождение`: раунд немедленно завершается, выбывшие получают ПО по тексту события.
- `Захват трона`: раунд немедленно завершается, игрок, вытянувший событие, получает 1 ПО.

## 16) Конец раунда

- Если в игре остался 1 игрок — раунд завершён.
- Если взята последняя карта колоды:
- текущий ход доигрывается;
- затем идёт завершение раунда и сравнение.
- Для ничьих при стандартном завершении (по колоде) используется согласованный tie-break:
- сравнение суммы сброса;
- если снова равенство — все игроки получают по 1 ПО.

## 17) Возврат карт в игру (события)

Есть разные алгоритмы, определяется текстом конкретного события.

### Кейс A: групповой возврат
- Каждый игрок выбирает 1 карту из своего сброса.
- Выбранные карты + отложенная карта раунда замешиваются в колоду персонажей.
- Если у игрока пустой сброс, он ничего не делает.

### Кейс B: возврат при выбывании
- Выбывший выбирает 1 карту из своего сброса.
- Дополнительно берутся до 2 карт с верха колоды (пока карты есть).
- Эти карты замешиваются и кладутся обратно наверх колоды.
- Если колода пуста, дополнительный добор не делается.

### Приоритет
- Применяется именно тот алгоритм, который явно указан у конкретного события.
- Эффекты событий приоритетны.

## 18) Лимит активных событий в раунде

- В раунде может быть до 2 активированных событий.
- Первое событие активируется картой `Знамение`.
- Второе событие может быть активировано эффектом указа `Братоубийство`.

## 19) Буря

- Собираются карты рук оставшихся игроков + отложенная карта.
- Колода перемешивается, игрокам раздаётся по 1 карте.
- Оставшаяся карта убирается из игры.
- Временные эффекты и жетоны на картах сбрасываются.
- По согласованному правилу: эффекты указов и событий при этом не сбрасываются.
